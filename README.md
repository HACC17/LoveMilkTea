### With the Ionic CLI:

```bash
$ sudo npm install -g ionic cordova
```

After cloning the repo, cd into the project directory:

Install Firebase:

```bash
$ npm install firebase angularfire2 --save
```

Run the app:

```bash
$ ionic serve --lab
```
